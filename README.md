# Fundamentos de Desarrollo Web

## Resumen del Curso

Este curso de 1 hora introduce a estudiantes de secundaria a la creación de sitios web sencillos con HTML y CSS. **¡No requiere descargas!** Solo abre tu navegador web.

**Duración total**: ~60 minutos (dividido en 7 lecciones)

**Público objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación

**Prerrequisitos**: Ninguno

**Herramientas necesarias**: ¡Solo tu navegador web! (Chrome, Firefox, Safari, Edge) y el editor online Auladiv.

**Repositorio de código fuente**: [COMENTARIO: Enlace al repositorio público de GitHub, llamado webdev-course-[nombre del equipo]]

## Secuencia de la lección

### Lección 1: HTML y CSS, ¿Cómo se Construye la Web? 

**Descripción**: Aprende de qué están hechos los sitios web (analogía de la casa) y por qué HTML y CSS son los componentes básicos. Presentación del entorno Auladiv.

**Enlace**: [Ver la lección](COMENTARIO: Enlace al video de YouTube)

**Consejos clave**: **HTML** = Estructura (Esqueleto); **CSS** = Estilo (Pintura).

**Práctica**: [Abre Auladiv](https://www.auladiv.com/ediweb/) ¡Empieza a programar!

### Lección 2: Estructura Básica y Conexión CSS

**Descripción**: Explora la **Estructura Canónica** de HTML ("html", "head", "body"), la conexión de los estilos (<link>), y crea los elementos iniciales ("header", "h1", "p").

**Enlace**: [Ver la lección] (COMENTARIO: Enlace al video de YouTube)

**Consejos clave**: Todo el contenido visible va en el <body>. Usa **<link>** en el <head> para conectar CSS.

**Práctica**: [Clic para programar](https://www.auladiv.com/ediweb/) Si te pierdes, usa el archivo `starter-files/L2.html` de GitHub.

### Lección 3: Contenido Enriquecido: Listas, Imagen y Anclas 

**Descripción**: Agrega listas ("ul"), imágenes ("img"), y enlaces ("a") para que tu página sea interactiva. Aprende a usar **Anclas** (`id` y `href="#id"`) para la navegación interna.

**Enlace**: [Ver la lección](COMENTARIO: Enlace al video de YouTube)

**Consejos clave**: Usa siempre el atributo `alt` en `<img>`. Los `id` y `href` de ancla deben coincidir.

**Práctica**: [Clic para programar](https://www.auladiv.com/ediweb/) Si necesitas ayuda, descarga `starter-files/L3.html` de GitHub.

### Lección 4: CSS I: Estilos Globales y Texto

**Descripción**: Dale estilo global a tu HTML (fondo, tipografía, color) y aplica estilos a los títulos (`h1`, `h2`) usando selectores de etiqueta y colores hexadecimales.

**Enlace**: [Ver la lección](COMENTARIO: Enlace al video de YouTube)

**Consejos clave**: La sintaxis es `selector { propiedad: valor; }`. Usa **`padding-top: 60px;`** para evitar que la barra superior oculte el contenido.

**Práctica**: [Aplicar estilo con un clic](https://www.auladiv.com/ediweb/) Usa el código final de la Lección 3 para iniciar.

### Lección 5: CSS II: El Modelo de Caja (Box Model) 

**Descripción**: Concepto del **Modelo de Caja** . Aprende a usar un contenedor (<div>) para centrar el perfil (`margin: auto;`), aplicar relleno (`padding`) y transformar la imagen a un círculo (`border-radius: 50%`).

**Enlace**: [Ver la lección](COMENTARIO: Enlace al video de YouTube)

**Consejos clave**: **`margin: auto;`** centra la caja horizontalmente. **`padding`** es el espacio _interno_ de la caja.

**Práctica**: [Aplicar estilo con un clic](https://www.auladiv.com/ediweb/) El código base está en `completed-examples/L4.css` en GitHub.

### Lección 6: CSS III: Construyendo la Toolbar (Flexbox)

**Descripción**: Posicionamiento avanzado. Fija la barra de navegación con **`position: fixed`**. Aprende a alinear elementos de forma moderna usando **Flexbox** (`display: flex;` y `justify-content`).

**Enlace**: [Ver la lección](COMENTARIO: Enlace al video de YouTube)

**Consejos clave**: **`position: fixed`** mantiene la barra visible al hacer _scroll_. **`justify-content: space-between;`** separa el logo del menú.

**Práctica**: [Crea tu Toolbar](https://www.auladiv.com/ediweb/) El código base está en `completed-examples/L5.css` en GitHub.

### Lección 7: Interactividad y Conclusión 

**Descripción**: Toques finales y próximos pasos. Se añade interactividad con la pseudoclase **`:hover`**. Se agrega el efecto de **Scroll Suave** (`scroll-behavior: smooth`) y se dan las recomendaciones finales.

**Enlace**: [Ver Lección] (COMENTARIO: Enlace al video de YouTube)

**Consejos clave**: **`:hover`** añade interactividad. **Error Común:** Revisar siempre las etiquetas de cierre.
